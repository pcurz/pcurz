## 🔭 **Projects I've Been Involved In**
- **📅 [UCalendarMobile](https://play.google.com/store/apps/details?id=dev.pcurz.ucmobile)**  
    A **mobile app** that provides students with easy access to the **Universidad Católica del Maule's academic calendar**. The app **fetches real-time updates** from the official site to ensure users always have the latest information. Now available on the **Google Play Store**!
- **🌐 [GTA Speedrun LATAM Racing](https://github.com/ferecece/gtasrlatamracing-leaderboard)**  
   A **real-time tracking website** for a speedrunning community with **600+ members**, using **Next.js** and a custom API connected to the Multi Theft Auto server.  
- **📲 [FallWatcher](https://github.com/pcurz/FallWatcher)**  
   A **real-time IoT system** for fall detection using ESP8266 devices, an MQTT broker, and a Flutter app. I orchestrated the **device communication** and **fall detection logic**, working with teammates who handled the physical device development.
- **⚡ [VaiaFlashcardsUploader](https://github.com/pcurz/VaiaFlashCardsUploader)**  
   An **automation tool using Puppeteer** to upload educational flashcards faster on Vaia, saving hours of manual work.
